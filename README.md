# Beacon을 이용한 출결 관리 시스템
## 졸업 작품 요약
- 학생들의 부정 출석 방지를 위한 기존 시스템의 보안 기능 강화
- 학생들의 편의성 향상을 위한 기존의 시스템 보완 및 새로운 기능 추가
- IoT 기술(비콘)을 활용한 기존 시스템의 편리성 및 효율성 추가

## 기능
- 회원가입, 로그인 기능
- 수강, 출석, 조회 관련 기능
- 개인정보 수정 기능
- 비콘 통신 기능

## 사용된 기술 또는 라이브러리
- Spring Boot : Jpa, Spring Security
- MySQL
- AltBeacon 라이브러리: 스마트폰과 비콘간의 통신 
- Retrofit 라이브러리: HTTP API 통신
